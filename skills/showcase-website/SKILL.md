# Showcase Skill (Stable)

Veröffentlicht HTML-Dateien im Browser-Showcase.

## Befehle (Absolute Pfade)
```bash
python3 /home/node/.openclaw/workspace/skills/showcase-website/scripts/expose_showcase.py <HTML_DATEI> [--name custom.html]
```

## 🧠 Model-Instruction
1. HTML-Datei muss sich im Workspace befinden.
2. Nutze den absoluten Pfad zum Skript.
3. Poste den Link: `✅ Showcase: https://claw.raspb.eu/showcase/<dateiname>`.
